Polygon Filling Applet:
-----------------------

This applet shows how to check if a polygon is convex and how to fill an arbitrary polygon with a scanline alogrithm. It refers to the article on my site 'Polygon Filling'.


How to run the applet:
-----------------------
Just open the file PolygonApplet.html in /build directory in your preferred browser.
